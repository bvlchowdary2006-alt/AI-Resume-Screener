# React + Vite (JavaScript)

This is a production-grade frontend for the AI-Powered Resume Screening System.

## Features
- Built with React and Vite
- Styled with Tailwind CSS
- Iconography by Lucide React
- Visualizations by Recharts
- Animations by Framer Motion
- API communication via Axios

## Getting Started
1. `npm install`
2. `npm run dev` for local development
3. `npm run build` for production build
